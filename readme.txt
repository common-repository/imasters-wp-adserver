== Description ==

iMasters WP AdServer is used to manage ads in your WordPress based site. A complete support to manage ads with date expiration controll, clicks and impressions statistics.

== Installation ==

How to install iMasters WP AdServer

e.g.

1. Upload `imasters-wp-adserver` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the plugin through the 'iMasters WP AdServer' menu in WordPress
